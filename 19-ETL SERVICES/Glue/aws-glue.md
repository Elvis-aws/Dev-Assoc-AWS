
# AWS Glue
- ETL service
- Serverless service
- Provides Crawlers: They automatically
    - Discover all your data sets
    - Discover your file types
    - Extract the schema
    - Store all this information in a meta-data glue catalogue
- It generates the script you need for your ETL job
